# (バス比較なび)['https://www.bushikaku.net/']から、情報をスクレイピングするアプリです

***

### 取得できる情報
- eventdates : バスの運行日
- data_get_dates : 今日の日付
- from_where : バスの出発地
- from_where_details : バスの出発地の詳細
- from_where_times : バスの出発時間
- to_where : バスの到着地
- to_where_details : バスの到着地の詳細
- to_where_times : バスの到着時間
- prices : そのバスの複数プランの中の最低価格
- bus_cos : そのバスの運営会社
